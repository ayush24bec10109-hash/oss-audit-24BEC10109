## Script Descriptions

Script 1 - system_info.sh
  Displays username, hostname, uptime, kernel version, and OS name.

Script 2 - package_checker.sh
  Checks if Git is installed. Shows version and location if found.

Script 3 - disk_auditor.sh
  Shows disk usage and file permissions of key system directories.

Script 4 - log_analyzer.sh
  Reads a log file and counts error, warning, and failed messages.

Script 5 - manifest_generator.sh
  Takes user input and saves a structured project manifest file.
